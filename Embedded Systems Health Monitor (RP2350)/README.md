Full Description
----------------

An embedded health-monitoring system built on the RP2350. Reads real-time heart rate and temperature using I2C sensors, displays the values, and triggers audible alerts when thresholds are exceeded.

Key Features
------------

- I2C communication with sensor module
- OLED/LCD output for vitals
- Threshold-based buzzer alarms
- Bare-metal C implementation (no RTOS)
- Validated accuracy vs commercial trackers

Tools
-----

C, RP2350 SDK, I2C debugging, Logic Analyzer
